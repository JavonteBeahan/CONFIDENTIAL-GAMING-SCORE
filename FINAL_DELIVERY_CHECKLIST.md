## How to Use This Delivery

### For Quick Start
1. Review `README.md` - Main documentation
2. Review `TUTORIAL.md` - Beginner introduction
3. Run: `npm install && npm run test`

### For Development
1. Review `DEVELOPMENT_GUIDE.md` - Code standards
2. Review contract source code
3. Review test files for patterns
4. Modify as needed

### For Creating New Examples
1. Review `ADDING_EXAMPLES.md` - Contribution guide
2. Create contract in `contracts/`
3. Create test in `test/`
4. Run: `npx ts-node scripts/create-fhevm-example.ts your-example ./output`

### For Deployment
1. Review `DEPLOYMENT_GUIDE.md` - Network instructions
2. Set up `.env` with private key
3. Run: `npm run deploy:zama`
