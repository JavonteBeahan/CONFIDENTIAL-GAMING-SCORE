## Getting Started

### Quick Start (5 minutes)
```bash
cd AnonymousGamingScore
npm install
npm run compile
npm run test
```

### Generate Your Own Example (2 minutes)
```bash
npx ts-node scripts/create-fhevm-example.ts fhe-counter ./my-example
cd my-example
npm install
npm run test
```

### Deploy to Network
```bash
cp .env.example .env
# Edit .env with your private key
npm run deploy:zama
```
